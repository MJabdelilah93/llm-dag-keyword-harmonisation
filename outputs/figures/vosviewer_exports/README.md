# outputs/figures/vosviewer_exports/ — Restricted

This directory previously contained co-occurrence matrices, edge lists,
and keyword frequency files for VOSviewer visualisation. These files
contain keyword strings derived from the Scopus corpus.

**Access:** Available in the restricted Zenodo dataset record:
https://doi.org/10.5281/zenodo.19451886

**Regeneration:** After obtaining the corpus (see `docs/reproducibility.md`),
run `python scripts/export_vosviewer.py` to regenerate these files.
