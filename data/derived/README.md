# data/derived/ — Restricted

This directory previously contained keyword frequency tables derived from
the Scopus corpus. Although these are aggregate counts rather than raw
records, the keyword strings themselves are Scopus-derived metadata and
are not publicly redistributable under Elsevier's Terms of Use.

**Access:** Available in the restricted Zenodo dataset record:
https://doi.org/10.5281/zenodo.19451886

**Corpus statistics** (no keyword strings):

| Statistic | Value |
|-----------|-------|
| Total author keyword occurrences | 141,284 |
| Unique author keyword strings | 55,425 |
| Singleton keywords (f = 1) | 42,913 (77.4%) |
| Keywords with f ≥ 5 | 3,648 (6.6%) |
| Unique index keyword strings | 62,988 |

See Table 2 of the manuscript for the full corpus and keyword inventory.
